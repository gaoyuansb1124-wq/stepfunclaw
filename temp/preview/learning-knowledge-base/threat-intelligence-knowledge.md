# 威胁情报写作手法学习笔记

> 学习素材：（玉林）美伊空袭叠加伊朗遭遇史上最大规模网络攻击 为我方网络空间防御敲响警钟
> 学习时间：2026-04-01
> 核心价值：被上级部门采纳的威胁情报专题报告范本
> 存放位置：skills/learning-knowledge-base/threat-intelligence-writing.md

---

## 一、文章结构分析

```
标题：事件核心 + 警示意义（二句话式）
         ↓
概览段：背景 + 时间 + 发起方 + 核心判断（一段式，约300字）
         ↓
一、攻击战术：AI深度赋能，打造现代网络战全新范式
         ↓
二、作战逻辑：网武先行，构建网络与物理一体的协同打击链
         ↓
三、攻击目标：直击关键基础设施，动摇国家运转根基
         ↓
四、对我国威胁分析
    （一）美网络霸权实操升级，我国核心领域面临技术复刻风险
    （二）供应链安全短板凸显，我国境外依赖领域存在渗透隐患
    （三）网络空间对抗泛化，我国海外利益与国际话语权遇挑战
         ↓
五、应对策略
    下一步应立即开展以下工作...
```

**关键发现：** 不同于标准公文格式的"一、基本情况 / 二、事件影响 / 三、对策建议"，这篇用了更贴合热点的自定义标题，像专题研究报告。

## 二、切入角度总结

### 角度1：以终为始——先写对我国的具体威胁
不是先详述事件本身，而是从"这事对我国有什么威胁"切入，引发上级部门高度关注。
- 具体写法：把"美可能将此战法复刻到我国"作为核心论点

### 角度2：层次分明的威胁推导
（一）→ 美军直接威胁：技术复刻到我国核心领域
（二）→ 供应链隐患：能源/工控/通信依赖外国产品被渗透
（三）→ 地缘政治：网络对抗泛化，海外利益受损

### 角度3：攻击逻辑写"新"，威胁分析写"我国"
- 攻击部分：突出 AI Agent、供应链预置、网电一体协同——强调"新"
- 威胁部分：直接对应我国能源、电网、通信、卫星等具体系统——强调"我国"

## 三、可学习的写作技巧

### 1. 概览段写法
```
"2月28日，美以'史诗狂怒行动''狮吼行动'对伊朗发动史上规模最大、
破坏力最强、协同程度最高的协同网络攻击，将网络战作为作战发起与
核心环节...攻击呈现全国断网、陆域空间混合、长期潜伏三重特征..."
```
- 用时间开头锚定事件
- 用"史上规模最大、破坏力最强"定性，建立紧迫感
- 用技术特征（全国断网、混合、潜伏）引导读者预期

### 2. 对策建议段写法
```
"下一步，我方应立即开展以下工作：
一、继续加大舆情监测和情报侦察力度...做好应对反制准备；
二、加快关键基础设施网络防御体系建设，推进核心系统自主可控，
    构建平战一体的网络安全应急响应机制..."
```
- 以"下一步，我方应立即开展以下工作"开头，干脆有力
- 不说空话，每条都有具体动作（舆情监测、自主可控、应急响应）

### 3. 结尾收束写法
```
"标志网络战已成为国别间冲突的'撒手锏'，其作战模式与战术手段
为全球网络安全格局带来全新挑战，也为我国网络空间防御敲响警钟"
```
- 用"标志...已成为..."做战略定性
- 用"为...敲响警钟"呼应标题，形成闭环

## 四、核心表达库

| 场景 | 表达 |
|------|------|
| 攻击者技术强 | "AI深度赋能...突破传统边界与作战效能" |
| 攻击规模大 | "史上规模最大、破坏力最强、协同程度最高" |

| 对我威胁-技术 | "技术复刻风险...定向渗透与精准打击" |

## 六、瑶瑶的写作改进方向

1. **标题**：加入"为我方敲响警钟"等对我国警示性表述
2. **概览**：先说事件影响力和规模，再展开技术细节
3. **威胁分析**：每个点都要落到中国的具体系统/行业上，不能停在"可能对我国有威胁"的套话
4. **对策建议**：每条建议都要有具体可执行的动作，不能只说"加强安全意识"
5. **结尾**：用战略定性句呼应标题，形成完整闭环
