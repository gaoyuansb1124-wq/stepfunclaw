# 踩坑知识库 README

记录历史上踩过的坑、错误经验和解决方法。
每次出现新问题，第一时间补充进来，避免重复掉坑。

## 分类

| 文件 | 内容 |
|------|------|
| browser.md | 浏览器与网页抓取 |
| api-calling.md | API 调用与接口对接 |
| skill-install.md | 技能安装与配置 |
| tools.md | 工具使用（PowerShell/Python/命令行等）|

## 使用规则

- 每次遇到新坑 → 立即记录，不要等到"下次再说"
- 每次开始相关任务前 → 先读对应文件，预防已知问题
- 格式统一：**坑描述 → 根本原因 → 正确做法**
