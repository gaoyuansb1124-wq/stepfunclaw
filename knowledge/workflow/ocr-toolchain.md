# 图片/文档 OCR 工具链

更新日期：2026-04-30

## 分层工具链

```
输入类型
├── 图片（含文字）
│   └── 优先：PaddleOCR（百度，中文最强，本地免费）
│
├── PDF（扫描版/文字版）
│   └── 优先：PDF to Markdown（支持OCR+章节拆分+目录，本地免费）
│
└── 音频/视频/其他多模态
    └── 优先：多模态处理器（本地免费，覆盖全场景）

↓ 以上工具效果不佳时

升级方案：阶跃星辰视觉 API（质量最高，消耗token额度）
```

## 决策原则

1. 按输入类型匹配对应工具
2. 本地工具优先，零成本
3. 本地工具效果不满意 → 升级阶跃星辰视觉API
4. 处理结果 → 转成Markdown → 走文本处理流程
